1.카테고리 리스트 출력기능
    -(o)등록되어 있는 카테고리명들이 출력되어야함
    -(o)카테고리에 몇개의 메모가 저장되어 있는지 개수가 출력되어야 함.
    -(o)카테고리 이름을 클릭하면 메모 리스트가 출력되어야함
2.카테고리 등록 기능
    -(o)카테고리 이름을 입력하여 등록하도록 함.
    -(o)새로 등록한 카테고리명을 확인할 수 있어야 함.
3.카테고리 수정 기능
    -(x)카테고리 이름을 수정할 수 있어야 함.
    -(x)수정된 결과를 카테고리 리스트에서 확인할 수 있어야함.
4.카테고리 삭제 기능
    -(x)선택한 카테고리를 삭제할 수 있도록 함.(삭제할 카테고리에 메모들이 저장되어 있더라고,해당 카테고리내의 모든 메모들과 함께 살제되야함)
    -(x)카테고리를 삭제할 때는 사용자에게 정말로 카테고리를 삭제할지의 여부를 확인해야함.
5.메모 리스트 출력 기능
    -(o)카테고리 리스트에서 카테고리 이름을 클릭하였을 때 수행되는 기능
    -(o)등록되어 있는 메모들에 대한 리스트(목록)가 출력되어야함.
    -(o)리스트에 출력되는 항목은 메모번호,메모제목,중요여부(별표 또는 다른 블릿기호),작성일로 함.
    -(x)메모 리스트의 각 메모 배경색은 메모 등록시에 설정한 배경색으로 출력되어야함.
    -(o)메모 리스트에 출력된 메모 제목을 선택하면, 상세한 메모 내용이 출력되어야함.
6.메모 등록기능
    -메모 등록시에 입력되어야 하는 항목:
        -(o)메모번호
        -(o)메모 제목
        -(o)중요여부
        -(o)메모 내용
        -(o)메모 배경색
        -(o)첨부 그림
        -(o)작성일
    -(o)메모번호와 작성일은 사용자 입력없이 자동으로 등록되어야 함.
7.메모 수정기능
    -(o)등록되어 있는 메모의 내용이 출력되어야함.
    -등록된 메모를 수정할 수 있어야 함
        -(o)메모 제목
        -(o)중요여부
        -(o)메모내용
        -(o)메모 배경색
        -(o)첨부그림
    -(o)작성일은 수정한 일시로 자동 변경 
8.메모 내용 검색기능
    -(x)사용자가 키워드를 입력하여 검색버튼을 클릭하면,등록된 모든 메모 리스트에서 검색어가 포함되어 있는 메모 리스트를 출력하는 기능
    -(x)검색된 메모 리스트의 제목을 클릭하면 해당 메모의 상세 내용이 출력되어야함
9.UI 네비게이션 구성
    -(x)사용자의 편의를 고려하여, 링크가 용이하게 연결되도록 네비게이션과 화면 UI/UX가 구성되어야함.
